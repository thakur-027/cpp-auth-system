How to Build & Run:

Prerequisites:
- A C++17-compatible compiler (like 'g++' or MSVC)
- VS Code / any terminal

Steps:

1. Compile the project:

- g++ system1.cpp securehash.cpp -o system1

2. Run the executable:

- ./system1        # On Linux/macOS
- system1.exe      # On Windows
